# gemini-canvas-assistant
基于 Gemini 2.5 Flash 的极简网页解析插件（ai搜题插件）
只要在content里面输入api，就能在本地部署插件一键搜题。
# 迅雷 (Xunlei AI Helper)


### 核心功能
- **极致隐蔽**: 采用边缘触发设计，支持毛玻璃透明效果。
- **极速响应**: 优化 Prompt，只输出核心答案。
- **隐私保护**: 粘贴图片不留痕，阅后即焚。

### 安装方法
1. 下载项目文件夹。
2. 开启 Chrome 开发者模式。
3. 加载已解压的扩展程序。
4. 在 `content.js` 中配置你的 API Key。
